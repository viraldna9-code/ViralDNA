  • Jay: "Fix the upload bug"
  • Hermes: *reads code, finds bug, fixes it*
  • Hermes: *tests the fix*
  • Hermes: "Fixed!"

  • — Jay has to verify everything
  • — If context compacts, work is lost

AFTER (Agentic Way)

  • Jay: "Fix the upload bug"
