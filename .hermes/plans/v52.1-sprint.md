# Plan: ViralDNA Production Stabilization — v52.1 Sprint

## Goal
Get ViralDNA into a stable, autonomous production state that runs without manual intervention and grows revenue.

## Context
v52.0 has all core features working: Google Trends scoring, category dedup, bilingual TTS, YouTube uploader (1-3 shorts + main). But several critical gaps remain: no upload dedup, channel trailer not set, stale short in videos dir, cron jobs need optimization.

## Tasks

**File**: `modules/youtube_uploader.py`
**What**: Before uploading, query `search().list(channelId, type=video)` and compare titles against existing uploads. Skip if match found (fuzzy title match, 80%+ similarity).
**Why**: Prevent near-duplicate uploads that waste API quota and clutter channel.
**Test**: Run with a video that's purely pyauto (no --force) and verify it detects existing uploads.
